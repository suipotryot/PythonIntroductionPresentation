# Python Introduction Presentation

## Requirements

1. Install vim
2. In vim, install posero (https://github.com/alfredodeza/posero.vim)

## Usage

Open demo.posero and start the presentation with:
```
:Poseron demo.posero
```

- C^J : step forward in slide
- C^K : step back in slide
- C^L : next slide
- C^H : previous slide
